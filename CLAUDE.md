# MVP for Account Heat Map web application

## Business Requirements
- User can sign in
- When signed in, the user sees a board that has Accounts going down the left side, and ten technology types going across the top
- Both the Accounts and Technologies can be renamed
- Accounts can be added
- Each account can have information entered into all the columns in its row for the technologies (e.g. AI Tools, AI Strategy, Data Governance, etc)
- The Account row and all its information can be dragged and dropped to other rows

## Limitations
For the MVP, there will be a user sign in (hardcoded to 'user" and 'password')
For the MVP, there will only be 1 account heat map board
For the MPV, this will just run locally
